# Для запуска тестов:

### 1. Сбилдить образ сервиса в fastapi-solution/ запустить: 
`docker-compose build`
### 2. Далее можно запускать контейнер с тестами, в fastapi-solution/tests:
`docker-compose up`

### 3. URL Swagger:
`http://127.0.0.1:8000/api/openapi#/`
